How to open and modify this repo

From a terminal:

  cd /Users/jingzhan/mech.v-rocq

Enter the Rocq/mathcomp shell:

  nix-shell

Check the active tools:

  rocq --version
  coqc --version
  dune --version

Build from inside the shell:

  dune build

Open an editor from the reproducible shell with Emacs and Proof General:

  nix-shell --arg withEmacs true --run "emacs mech/lib.v"

or open the repo root:

  nix-shell --arg withEmacs true --run "emacs ."

The first Emacs/Proof General run may take several minutes because Nix can need to build or download editor packages.

RocqIDE note:

- I checked the current Nix shell and it provides rocq and coqc, but not a GUI rocqide or coqide binary. It does provide the command-line Coq/Rocq tools needed by Dune and editor integrations.
- If a future Nix package set exposes RocqIDE/CoqIDE, try one of:

  nix-shell --run "rocqide mech/lib.v"
  nix-shell --run "coqide mech/lib.v"

- For this repo today, use Emacs + Proof General through nix-shell --arg withEmacs true.
